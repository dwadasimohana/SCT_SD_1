# SCT_SD_1: Temperature Converter

A simple, interactive, and robust command-line application written in C that converts temperatures between **Celsius**, **Fahrenheit**, and **Kelvin**. This project demonstrates the clean use of modular programming, functions, and user-input validation in C.

---

## 🚀 Features

- **Multi-unit Conversion:** Supports all six major conversion combinations:
  - Celsius to Fahrenheit & Kelvin
  - Fahrenheit to Celsius & Kelvin
  - Kelvin to Celsius & Fahrenheit
- **Modular Codebase:** Built using distinct C functions for each conversion mathematical formula to ensure code reusability.
- **User-Friendly Interface:** Clear terminal menus with input prompts.

---

## 🛠️ Technologies Used

- **Language:** C (C11 standard)
- **Compiler:** GCC (or any standard C compiler)
- **Environment:** Command Line Interface (CLI)
